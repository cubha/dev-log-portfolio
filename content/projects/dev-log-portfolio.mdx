## 프로젝트 개요

Next.js 15 App Router와 Supabase를 기반으로 구축한 개인 포트폴리오 사이트입니다.
단순 소개 페이지를 넘어 실시간 GitHub 활동, Spotify Now Playing, 방명록, 관리자 CRUD 대시보드까지 갖춘 풀스택 프로젝트입니다.

## 핵심 구현 내용

### 1. GitHub Live Status (GraphQL API)

GitHub REST Events API 대신 GraphQL `contributionsCollection`을 사용하여 Private 레포 커밋까지 포함한 정확한 기여도 집계를 구현했습니다.

- `repo` + `read:user` 스코프 PAT 발급
- `contributionsCollection.totalCommitContributions` 필드로 연간 커밋 수 조회
- 서버사이드 전용 호출로 토큰 클라이언트 노출 방지

### 2. Supabase 인증 및 관리자 시스템

- Supabase Auth 이메일/패스워드 로그인 → JWT 세션 발급
- `profiles` 테이블 `role` 컬럼으로 관리자 권한 분리
- Next.js 15 Middleware에서 JWT 검증 + role 기반 라우트 가드
- GitHub OAuth 연동 (방명록 소셜 로그인)

### 3. 프로젝트 CRUD 관리자 대시보드

- 관리자 전용 프로젝트 등록/수정/삭제
- Supabase Storage 이미지 업로드 (드래그 앤 드롭 지원, 5MB 제한)
- `detailed_tasks` 동적 리스트 입력 (`|||` 구분자 배열 처리)
- `project_type` 기반 LIVE DEMO 버튼 조건부 노출

### 4. 방명록 (Guestbook)

- 익명 / GitHub OAuth / 관리자 세 가지 인증 모드
- Supabase RLS(Row Level Security) 정책으로 삭제 권한 제어
- 페이지네이션 (10개/페이지), Server Action 기반 CRUD

### 5. 프로젝트 리스트 UI 고도화

- X축 드래그 슬라이더 → Y축 반응형 CSS Grid 전환 (`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`)
- Framer Motion `whileInView` Fade-up 애니메이션 (스크롤 진입 시 순차 노출)
- 카테고리 필터 (Jotai `projectFilterAtom`)
- 프로젝트 모달 → MDX 상세 페이지 라우팅

### 6. 3-AI 협업 개발 구조

- **Claude Code** — 컨트롤러 (분석·설계·프롬프트 작성·코드 검증)
- **Cursor** — 구현 전담
- **Gemini** — 외부 리서치 (라이브러리 비교, 트렌드 조사)

## 트러블슈팅

### 문제: GitHub Private 레포 커밋 집계 불가

**원인**: REST Events API(`/users/{username}/events`)는 Private 레포 `PushEvent`에서 `payload.size`, `payload.commits` 필드를 모두 제거(strip)함.

**해결**: GitHub GraphQL API의 `viewer.contributionsCollection.totalCommitContributions`로 대체. Private 포함 정확한 연간 커밋 수 반환.

---

### 문제: Supabase RLS SELECT 정책 무력화

**원인**: `USING(true)` 정책이 하나라도 존재하면 OR 합산으로 다른 제한 정책이 무효화됨.

**해결**: 공개/비공개 분리 정책 조합 방식으로 재설계. 단일 `USING(true)` 남용 방지.

---

### 문제: Framer Motion `whileInView` 신규 카드 미작동

**원인**: 이미 마운트된 부모 내에 새로 추가된 자식 컴포넌트는 부모 variant 전파를 받지 못함.

**해결**: 각 카드에 `initial` / `whileInView`를 명시적으로 직접 선언하여 독립 동작하도록 수정.
